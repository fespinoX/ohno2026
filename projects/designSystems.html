// Project Details

I built this design system from scratch in Figma and worked closely with developers to keep design and code in sync. Using the Tokens Studio for Figma plugin, we translated Figma variables into a JSON file that developers could process to automatically generate CSS variables for the React components. This setup helped keep the system consistent and easier to maintain as it evolved.
